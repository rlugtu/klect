# Klect — Follow-Along Production Guide

Two routes, both covered: the **cinematic route** (CapCut + Veo, for your four shot-planned ads) and the **UGC route** (Creatify/Arcads avatar testimonial ads). Run both, test them against each other.

Use this alongside `klect-video-shot-plans.md` — that doc has the full shot detail; this one is the click-by-click build.

---

## STEP 0 — One-time setup (~30–45 min, do this once)

**A. Build a beautiful demo account in Klect.** Every screen recording pulls from this, so it has to look great.
- Create one gorgeous public guide (e.g. "Best Coffee in [your city]") — 8–10 cards with real appetizing photos, star ratings, and a short note on each ("get the cortado").
- Create one collaborative list with 2–3 contributor avatars and an active poll.
- Set up varied card types on your home: a restaurant, a recipe, a product, a travel spot, and one with an inline video.
- Make sure tags are colorful and consistent.

**B. Install CapCut** (free, no watermark, mobile or desktop). This is your assembly + captions + b-roll hub. Sora 2 and Veo 3.1 are built into its AI generation panel.

**C. Prep screen recording.** On your phone: turn on Do Not Disturb, hide notifications, record at max resolution. Practice the "paste a link → card fills in" motion a few times so it's smooth.

---

## STEP 1 — The repeatable CapCut + Veo loop (reuse for all 4 videos)

Every video is the same 6-step loop:

1. **Generate the b-roll (AI-GEN shots).** In CapCut, open the AI video generation panel and select Veo 3.1. Paste the shot's prompt (provided per video below). Set 9:16, generate 2–3 variations per shot, keep the best. If CapCut's built-in credits run low, generate in a standalone Veo tool and import the clips.
2. **Record the screens (SCREEN REC shots).** Capture each app interaction from your demo account. Get 2–3 takes each; the "card auto-fills" shot is the money shot — get it clean.
3. **Lay the timeline in order.** Drop shots 1→5 on the timeline. Trim each to its 2–4s slot. Cut to the beat.
4. **Add on-screen text.** One bold line per shot (text provided below). Keep it in the middle 80% of the frame so platform UI doesn't cover it.
5. **Auto-caption + music.** Use CapCut auto-captions for the VO/spoken lines. Add music: trending audio for organic TikTok posts, licensed/royalty-free for paid ads.
6. **Export variants.** Export the full ad, then export **5 hook variants** — same video, swap only the first 3 seconds + opening text line. This is your #1 performance lever.

## STEP 3 — UGC avatar ad (Creatify or Arcads)

This is your second creative lane. These tools generate a realistic AI "creator" reading a script, so the ad looks like a real person's recommendation rather than a polished commercial — which is exactly why UGC-style ads often win for apps.

### How the workflow goes
1. Sign up (plans start ~$39/mo; both have free credits to test first).
2. **Add your app context** — paste your App Store URL or app description so the tool has the product.
3. **Choose an avatar** — pick one that matches your target user (e.g. a 20s–30s casual, relatable presenter, not a corporate spokesperson). Generate the same script with 2–3 different avatars to test.
4. **Paste your script** (below). Keep the tool's auto-script off — your hand-written scripts will outperform the generic ones.
5. **Set format:** 9:16 vertical, captions ON (burned-in), 15–30s.
6. **Insert your screen recordings** where the script references the app — most UGC tools let you cut to b-roll/screen clips over the avatar's voice. Drop in the "card auto-fills" and "shared guide" screens so viewers see the real product.
7. **Generate 3–5 variations**, export, and push to TikTok/Reels (both tools export native vertical + some integrate directly with Meta/TikTok).

### UGC script — Variant A (discovery / relatable)
> "Okay I have to tell you about this app because it fixed something that annoyed me for years. I save *everything* — restaurants, recipes, random stuff — and I could never find any of it again. This app is called Klect. You paste a link and it just… fills in the photo, the details, all of it. I tag it, rate it, and honestly the best part is I turned my saved restaurants into a guide I can send my friends. It's free. Just go get it."

### UGC script — Variant B (the recommendation friend)
> "If you're the friend everyone asks for recommendations, you need this. I put all my favorite spots into Klect — with photos, my ratings, little notes — and made it a public guide. Now instead of retyping the same five places in the group chat, I just send one link. Took me ten minutes. It's free on the App Store."

### UGC script — Variant C (group planning)
> "The way this app just saved our group trip. We were doing the usual — 200 messages, nobody agreeing, links getting lost. Then we put everything in one shared list on Klect, voted on stuff with actual polls, and it even shows you your saved spots near you. Planning was done in like a day. It's called Klect, it's free, go."

### UGC settings checklist
- [ ] 9:16 vertical
- [ ] Burned-in captions ON (most watch on mute)
- [ ] Hook lands in first 3 seconds — the first line above is the hook, don't bury it
- [ ] Real screen recordings composited over the app-mention lines
- [ ] 2–3 avatars × 3 scripts = a batch to test
- [ ] Truthful claims only, matched to real features

## STEP 4 — Publishing to TikTok + Instagram

- **Post organically first** on both, using trending audio, to see what resonates before spending.
- **Then run the winners as paid** — swap trending audio for licensed/royalty-free to avoid licensing issues in paid placements.
- **TikTok:** fast hook, native/UGC pacing, captions on.
- **Instagram Reels:** slightly more polished product-plus-lifestyle look plays well; same 9:16 assets work.
- **Test structure:** run 2 different *angles* against each other first (e.g. Video 1 pain vs the UGC discovery ad). Once one angle wins, test new hooks within it.
- **Judge on installs AND what happens after** — a cheap install that never builds a list isn't a real win.

## Quick reference — the stack
- **Assembly + captions:** CapCut (free)
- **AI b-roll:** Veo 3.1 (in CapCut) — or Kling 3.0 for the most filmed-looking shots
- **Cheap iteration:** Luma Ray (Draft Mode previews before spending credits)
- **Screen capture:** your phone's native recorder (free)
- **UGC avatar ads:** Creatify or Arcads (~$39/mo, free credits to start)
